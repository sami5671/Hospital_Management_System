# Hospital Management System

This is a full-stack Hospital Management System built using the MERN stack and Vite for the client-side setup.

## Features

- User Authentication
- Appointment Scheduling
- Patient Records Management
- Admin Dashboard

## Technologies Used

### Frontend

- React
- React Router DOM
- Axios
- React Toastify

### Backend

- Node.js
- Express
- MongoDB
